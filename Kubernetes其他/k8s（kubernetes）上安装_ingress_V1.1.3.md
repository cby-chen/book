介绍
==

Ingress 公开了从集群外部到集群内服务的 HTTP 和 HTTPS 路由。流量路由由 Ingress 资源上定义的规则控制。

下面是一个将所有流量都发送到同一 Service 的简单 **Ingress** 示例：

![](https://p3-juejin.byteimg.com/tos-cn-i-k3u1fbpfcp/bff59aa69c124a8a8e41a4514b4732ab~tplv-k3u1fbpfcp-zoom-1.image)
